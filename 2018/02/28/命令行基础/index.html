一. 常用命令

缩写 全写 命令
mkdir make directory 创建目录
touch 创建文件
echo echo “hi” > 1.txt 创建文件1.txt
echo “hi” >! 1.txt 覆盖1.txt
rm remove 删除
mv move 重命名
cp copy 复制
ls list 显示当前目录下所有文件
ls -a 显示隐藏文件
cd change directory 改变目录
ln -s 建立软链接

还可以把很多常见命令的缩写放进.bashrc里，比如

alias la=’ls -a’
alias ll=’ls -l’
alias gst=’git status -sb’
alias ga=’git add’
alias ga.=’git add .’
alias gc=’git commit’
alias gc.=’git commit .’

保存退出，然后运行 source ~/.bashrc
这样一来，你的 Git 操作就会简单很多
